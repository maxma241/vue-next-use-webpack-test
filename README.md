# vue-next-use-webpack-test

> Minimal webpack setup for Vue 3 (alpha) 

and try the composition api to creat `vue-use`

### Prerequisites
- Node & NPM

### Install
```sh
npm install
```
### Usage
##### Develop
```sh
# run dev server at localhost:8080
npm run dev
```
##### Build
```sh
# transpile js for deployment
npm run build
```
